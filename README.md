# Mentorly

## Used technologies
NodeJS v16.13.2  
MongoDB v4.2.17

## How to run

Run mongoDB on your local machine.  
Create duplicate file of `.env.example` with name `.env`.

And run commands below

`npm install`

`npm run dev`


## Postman Collections

In project directory you can find <b>Postman</b> API collections.
